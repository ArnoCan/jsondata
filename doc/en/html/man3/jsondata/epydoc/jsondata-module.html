Modular processing of JSON data by trees and branches, pointers and patches.

Submodules [hide private]
  • jsondata.JSONData: Core features for the processing of structured JSON based in-memory data.
  • jsondata.JSONDataExceptions: Common exceptions for the package 'jsondata'.
  • jsondata.JSONDataSerializer: Basic features for the persistence of JSON based in-memory data.
  • jsondata.JSONPatch: The JSONPatch module provides for the alteration of JSON data compliant to RFC6902.
  • jsondata.JSONPointer: Provides classes for the JSONPointer definition in accordance to RFC6901.
  • jsondata.JSONTree: The JSONTree module provides features for in-memory JSON structures.
  • jsondata.Selftest: Test of basic features for the user by '--selftest'.
  • jsondata.data_json
  • jsondata.jsondata: Core features for the processing of JSON based structures of in-memory data.

  • jsondata.jsondatacheck_json
  • jsondata.jsondataserializer: Basic features for the persistence of JSON based in-memory data.
  • jsondata.jsonpatch: The JSONPatch module provides for the alteration of JSON data compliant to RFC6902.
  • jsondata.jsonpointer: Provides classes for the JSONPointer definition in accordance to RFC6901 and relative pointers draft-1/2018.
